# Projeto de Desenvolvimento de Layout e Código de Site

## Introdução

Neste projeto, trabalhamos na criação de um site para um cliente, envolvendo desde o design do layout até a implementação do código. O projeto abrangeu todas as etapas do processo de UX, incluindo a concepção do layout, a construção do código HTML e o uso de SCSS para estilização.

## Etapas do Projeto

### 1. Concepção do Layout

Iniciei o projeto com a criação do layout visual, focando em oferecer uma experiência de usuário intuitiva e atraente. O processo envolveu:

- **Pesquisa e Análise:** Entendi as necessidades do cliente e o público-alvo para criar um design que atendesse às expectativas e objetivos.
- **Wireframes:** Desenvolvi wireframes para estruturar a disposição dos elementos na página e definir o fluxo de navegação.
- **Protótipos:** Criei protótipos interativos para testar e refinar o layout com base no feedback inicial.

### 2. Desenvolvimento do Código

Após a aprovação do layout, passei para a fase de codificação, que incluiu:

- **HTML:** Estruturei o conteúdo da página utilizando HTML, garantindo a semântica adequada e a acessibilidade.
- **SCSS:** Usei SCSS para estilizar a página, aproveitando seus recursos avançados como variáveis, aninhamento e mixins para criar um design responsivo e modular.

### 3. Processos de UX

Durante todo o projeto, concentrei-me em práticas de UX para garantir uma experiência de usuário de alta qualidade. As etapas incluíram:

![Alt Text](/images/thumbs/halsten-preview.png)

- **Design Responsivo:** Assegurei que o layout fosse otimizado para diferentes dispositivos e tamanhos de tela.
- **Testes de Usabilidade:** Realizei testes para identificar e resolver problemas de usabilidade, garantindo que a navegação fosse fluida e intuitiva.
- **Feedback e Iteração:** Recolhi feedback dos usuários e do cliente para fazer ajustes e melhorias contínuas no design e na funcionalidade.

## Conclusão

O projeto foi concluído com sucesso, entregando um site funcional e esteticamente agradável que atendeu às necessidades do cliente. O uso de SCSS e HTML permitiu a criação de uma página bem estruturada e estilizada, enquanto as práticas de UX garantiram uma experiência de usuário otimizada.

## Tecnologias Utilizadas

- **HTML:** Estruturação do conteúdo da página.
- **SCSS:** Estilização avançada e responsiva.
- **Ferramentas de Design:** Figma, Figjam.
